# docker-v

Neovim on Docker with extra configuration for the Fish Shell

## Requirements

- GNU/Linux
- Git
- Fish
- Docker
- GNU/Make

## Installation

```console
$ git clone https://github.com/aminnairi/docker-v.git
$ cd docker-v
```

## Build

```console
$ make build
```

## Fish Function

```fish
make fish
```

## Usage

```console
$ v
```
